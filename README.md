 FIFA World Cup Prediction Engine

A football prediction engine built using Python and Monte Carlo Simulation.

 Features

- Time-decayed historical weighting
- Competition weighting
- Neutral venue adjustment
- Elo rating adjustment
- Weighted Poisson model
- Monte Carlo simulation (100,000 simulations)
- Match win probabilities
- Predicted scorelines

 Tech Stack

- Python
- Pandas
- NumPy
- Google Colab

 Example

```python
predict_match("Portugal", "Spain")
```

Output:

- Portugal Win Probability
- Spain Win Probability
- Draw Probability
- Top predicted scorelines
